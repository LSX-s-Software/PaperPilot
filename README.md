# PaperPilot App

基于 SwiftUI 的多平台多人协作文献管理软件

## 核心功能

> 功能正在开发中

- 文献管理
  - 自动识别文献的元数据并归类
  - 通过文献的元数据查询文献
- 跨平台、跨设备访问
  - 在电脑、平板等设备上查看文献
  - 数据多端同步
- 多人协同阅读
  - **实时同步**论文批注
  - **实时同步**笔记
- AI 协助阅读
  - 使用大语言模型接口实现论文解析、翻译、总结

## 支持的平台

- macOS 13.0+
- iPadOS 16.0+

## 开发要求

- macOS 13.0+
- Xcode 15.0+
