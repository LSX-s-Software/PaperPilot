基于 SwiftUI 的多平台多人协作文献管理软件

## 核心功能

> 功能正在开发中

- [ ] 文献管理
  - [x] 通过文献的元数据查询文献
  - [ ] 自动识别文献的元数据并归类
- [x] 跨平台、跨设备访问
  - [x] 在电脑、平板等设备上查看文献
  - [x] 数据多端同步
- [ ] 多人协同阅读
  - [x] **实时同步**笔记
  - [ ] **实时同步**论文批注
- [ ] AI 协助阅读
  - [x] 论文划词翻译
  - [ ] 论文解析、总结


## 支持的平台

- macOS 14.0+
- iPadOS 17.0+

## 开发要求

- macOS 14.0+
- Xcode 15.0+
